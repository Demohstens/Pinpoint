### Pinpoint

Adds a simple coordinate overlay that helps _pinpoint_ your location. Might add future config options and debug features similar to F3

This uses the deprecated "getPlayers" method as I could not for the life of me figure out how to get all players
otherwise. Waiting for updated docs to improve this.

I used DarkhaxDev's Overstacked mod as a "template" so shotout where it's due! Go check out [their stuff](https://www.curseforge.com/members/darkhaxdev/projects)
